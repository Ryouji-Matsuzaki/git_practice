### ■文字列

PHPで文字列をする場合、以前もありましたが

* シングルクォーテーション『'』
* ダブルクォーテーション『"』

のどちらかで文字を囲む必要があります。

そして、『"』と『'』は文字列に変数が含まれた場合に、処理方法が変化します。

* 『'』の場合

  * 文字列内の変数は文字として扱う


* 『"』の場合

  * 文字列内の変数は中身に置き換える


```php
<?php
$sample = "サンプル";

// シングルクォーテーションの場合、
// $sampleは文字として扱われるので、
// 「このプログラムの変すは、$sampleです。」と表示される
echo 'このプログラムの変数は、$sampleです。';

// ダブルクォーテーションの場合、
// $sampleが中身に置き換わるので
// 「このプログラムの変数は、サンプルです。」と表示される
echo "このプログラムの変数は、$sampleです。";

```
PHPで文字列を連結する場合、『.』を利用します。

また、『.』で連結できるのは文字列に限らず、数値なども文字として評価された上で連結されます。

** 他のプログラミング言語を勉強していると、文字の連結を『+』と思いがちですが、PHPでは『.』ですのでご注意ください。 **

```php
<?php
const NAME = '日本';
const NUM = 1;
$cheer = '頑張れ！';

// 日本昔話と表示されます
echo NAME . '昔話';

// 日本1と表示されます
echo NAME . NUM;

// 日本頑張れ！と表示されます
echo NAME . $cheer;
```
